# MVP – Engenharia de Dados | PUC-Rio

Este repositório contém o desenvolvimento do MVP da disciplina Engenharia de Dados
da Pós-Graduação em Ciência de Dados e Analytics – PUC-Rio.

O objetivo do trabalho é construir um pipeline de dados em nuvem utilizando a plataforma
Databricks Community Edition, contemplando as etapas de coleta, modelagem, carga,
análise e documentação dos dados.
