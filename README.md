# MVP – Engenharia de Dados — PUC-Rio

## Objetivo do MVP

O objetivo deste MVP é projetar e implementar um **pipeline de dados em nuvem** voltado à área de **investimentos no mercado financeiro brasileiro**, contemplando ativos de **renda variável (ações)** e **renda fixa**, desde a coleta de dados brutos até sua organização, armazenamento e análise.

O trabalho busca demonstrar competências em **Engenharia de Dados**, incluindo:

- ingestão de dados a partir de fontes públicas;
- tratamento e padronização de séries temporais financeiras;
- modelagem dos dados em um formato analítico adequado (Data Lake ou Data Warehouse);
- construção de pipelines de ETL em ambiente de nuvem;
- disponibilização dos dados para análises exploratórias e consultas analíticas.

Como estudo de caso, o MVP tem como foco **analisar o comportamento histórico, a variabilidade e o desempenho** de diferentes classes de ativos financeiros no Brasil, considerando **indicadores básicos de retorno e risco ao longo do tempo**, com o objetivo de **apoiar pessoas sem conhecimento prévio em investimentos** na escolha do tipo de ativo mais adequado, com base no desempenho observado nos últimos anos.
